# FORZA HORIZON

Félicitations pour votre achat de Forza Horizon !

## Récupération des accès

Contrairement à certains autres jeux, les mots de passe et identifiants pour FORZA HORIZON ne sont pas distribués publiquement dans un salon ou par le bot directement.

Une fois que votre achat est **vérifié** :
1. Un membre de notre équipe (le Staff) interviendra directement dans votre ticket.
2. Le membre de l'équipe vous fournira personnellement vos identifiants de connexion.
3. Vous recevrez également un bouton spécial dans le ticket pour générer ou recevoir le **Code 2FA (Authentification à double facteur)** nécessaire pour vous connecter au compte.

## Règles générales

- **Ne changez jamais le mot de passe :** Il est strictement interdit de modifier les informations de connexion du compte qui vous a été fourni.
- Conservez le compte dans l'état où il vous a été livré.

Si vous rencontrez un problème lors de l'activation ou si le code 2FA n'arrive pas, n'hésitez pas à le signaler directement dans votre ticket pour que le membre du staff puisse vous aider.
