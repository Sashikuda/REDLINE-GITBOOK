# ⤵️Downgrading Windows Versions (24H2)

Some of our products don't work with the most recent windows 11 version (24h2), this guide will show you how to downgrade successfully.

### 

[](#downgrade-from-windows-11-24h2-to-23h2)

Downgrade from Windows 11 24H2 to 23H2

#### 

[](#prerequisites)

Prerequisites

1.  Ensure that you have backed up all important data.
    
2.  Make sure your device is connected to the internet.
    
3.  Verify that your device is plugged into a power source.
    

#### 

[](#step-by-step-guide)

Step-by-Step Guide

1.  **Open Settings**: Press `Windows + I` to open the Settings app.
    

#### 

[](#important-notes)

Important Notes

*   This process should retain personal files but might uninstall some apps or settings configured after the 24H2 update.
    
*   If the `Go back` option is unavailable, you might need a fresh installation of Windows 11 23H2.
    

**Requirements**

*   A USB drive with at least 8GB of storage.
    
*   Windows 11 23H2 ISO file downloaded from the Microsoft website.
    
*   A tool like Rufus to create a bootable USB drive.
    

**Steps**

1.  **Create a Bootable USB Drive**:
    
    *   Insert your USB drive into the computer.
        
    *   Open Rufus and select your USB drive.
        
    *   Choose the downloaded Windows 11 23H2 ISO file.
        
    *   Click on 'Start' to create the bootable USB drive.
        
    
    **Important Notes**
    
    *   Ensure that all peripheral devices required for installation are connected.
        
    *   You may need to reinstall drivers and updates after downgrading.
        
    *   Verify that your device is compatible with the Windows 11 23H2 version.
