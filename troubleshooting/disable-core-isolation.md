# 🛡️﹒Disable Core Isolation

### 

[](#how-do-i-disable-core-isolation)

How do I disable core Isolation?

1.  **Open Start Menu**: Click on the Start button or press the Windows key on your keyboard.
    
2.  **Access Settings**: Click on the gear-shaped icon to open the Settings application.
    
3.  **Navigate to Update & Security**: In the Settings window, click on "Update & Security".
    
4.  **Open Windows Security**: From the left-hand menu, select "Windows Security".
    
5.  **Access Device Security**: Click on "Device security" in the list of Windows Security options.
    
6.  **Open Core Isolation Details**: Under Core isolation, click on "Core isolation details".
    
7.  **Disable Memory Integrity**: Toggle the switch to turn off "Memory integrity".
    
8.  **Restart Your Computer**: For changes to take effect, restart your computer.
