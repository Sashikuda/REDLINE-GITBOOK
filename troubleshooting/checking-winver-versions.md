How to Check Your Windows Version

Follow these steps to find out which version of Windows you are running:

1.  Press `Win + R` on your keyboard to open the Run dialog box.
    
2.  Type `winver` in the text field.
    
3.  Click `OK` or press `Enter`.
    

A window will appear displaying your Windows version and build number.

_**Why is this important?**_ Some of our products only target certain Windows Buids, most commonly Windows 11 24H2 and above might not have full compatibility compared to Windows 11 23H2 and below - Always read the product description!
