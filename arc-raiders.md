## Solutions Techniques (Dépannage)

Si vous rencontrez des problèmes, voici les solutions aux erreurs les plus fréquentes :

### 1. Problèmes de connexion généraux
Utilisez **Proton VPN** exclusivement pour le processus de connexion si vous n'arrivez pas à vous connecter.

### 2. Erreur 0x...
- Fermez complètement l'application.
- Utilisez un VPN localisé aux **États-Unis (USA)**.
- Assurez-vous que la région de votre système Windows correspond à la région du VPN (USA).

### 3. Demande de code PIN ("PIN Prompt")
Si l'application vous demande un code PIN, utilisez simplement **le mot de passe de votre propre ordinateur/session Windows**.

### 4. Crash "Pioneer"
Vous devez installer les pilotes NVIDIA spécifiques indiqués dans la documentation complète.

### 5. Erreur d'abonnement (Subscription Error)
Vérifiez que vous êtes bien connecté au bon compte Microsoft sur le Microsoft Store, et non sur votre compte personnel.
